# Proyecto 6: Generador de Landing Page con IA

Este proyecto demuestra la capacidad de generar una landing page completa y funcional para una empresa de consultoría ficticia ("Synergy Solutions") utilizando una serie de prompts detallados con un modelo de lenguaje avanzado.

## 🎯 Objetivo

El objetivo era simular un flujo de trabajo de "desarrollo asistido por IA", desde la creación del contenido de marketing (copywriting) hasta la generación del código HTML y CSS final.

## 🤖 Proceso

1.  **Generación de Contenido:** Se crearon prompts para definir la propuesta de valor, titulares, beneficios clave y testimonios de la empresa.
2.  **Generación de Código:** Se utilizó un "Super-Prompt" que contenía todo el contenido generado y especificaciones de diseño (tema oscuro, colores, tipografía, estructura responsiva) para que la IA generara el archivo `index.html` completo.

## 📂 Estructura del Repositorio

* `/prompts`: Contiene los archivos `.txt` con los prompts exactos utilizados.
* `/resultado`: Contiene el archivo `index.html` final generado por la IA y una captura de pantalla.
